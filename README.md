# Digital-Forensics-Cookbook

This repository contains the datasets and curated software download links used to support the hands-on exercises in the *Digital Forensics Cookbook* (Packt Publishing).

## 📘 How to Use This Repository

This repository is designed to accompany the hands-on exercises in the *Digital Forensics Cookbook*.

- Follow the exercises in each chapter of the book  
- Download the corresponding datasets as needed  
- Use the provided software tool links to replicate workflows  
- Validate findings using multiple artifacts and techniques where applicable  

## 📁 Repository Structure

- **Datasets** → Sample forensic images and lab data used in exercises  
- **Software Tools** → Curated links to official tool download sources  

## ⚠️ Data and Tool Usage

All datasets in this repository were created from controlled test environments and have been sanitized to remove any personally identifiable information (PII).

Software links are provided from official vendor sources or trusted open-source repositories. Tools should be independently validated prior to use in production or investigative environments.
