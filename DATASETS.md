# Datasets

This repository provides curated datasets used throughout the *Digital Forensics Cookbook* to support hands-on forensic analysis and investigative workflows.

---

## ⚠️ Data Usage & Requirements

All datasets were created from controlled test environments and have been sanitized to remove any personally identifiable information (PII).

- **Archive Size** → Size of the downloaded compressed file  
- **Uncompressed** → Required storage space after extraction  
- **Hashes (SHA256/MD5)** → Provided for integrity verification  

> Total uncompressed size for all datasets combined ≈ **55.6 GB**

> [!TIP]
> Hold the **CTRL** key when clicking links to open downloads in a new tab<br><br>
> Mirror links are provided as alternate download sources in case of platform limitations or availability issues  

> [!IMPORTANT]
> Ensure your system or storage device has sufficient space for extraction before downloading large datasets<br><br>
> Extract archives using [7-Zip](https://www.7-zip.org/download.html)
